# Struktur Data - Simulasi Routing Paket

Program ini dibuat menggunakan bahasa C++ untuk tugas mata kuliah Struktur Data.

Project ini merupakan simulasi sederhana pengiriman paket antar kota menggunakan beberapa struktur data.

## Struktur Data yang Digunakan
- Queue
- Directed Graph
- Linked List

## Fitur Program
- Tambah paket baru
- Kirim paket antar kota
- Lihat antrean gudang
- Cek histori perjalanan paket
- Lihat rute distribusi

## Penjelasan Struktur Data
- Queue digunakan sebagai antrean paket pada gudang
- Graph digunakan untuk menyimpan jalur distribusi antar kota
- Linked List digunakan untuk menyimpan histori perjalanan paket

## Bahasa Pemrograman
- C++

## File Program
- tugas_st.cpp

## Cara Menjalankan
1. Compile file tugas_st.cpp
2. Jalankan program melalui terminal
